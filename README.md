# Portfolio V1

A personal portfolio web application built to showcase projects, skills, and experience. Developed using **React** and **CSS**, this portfolio highlights professional work and personal projects in a modern and responsive design.

---

## Features

- Responsive layout for desktop and mobile devices  
- Showcase of projects with descriptions and links  
- Skills section with relevant technologies  
- Contact form to get in touch directly  
- Smooth navigation between sections
